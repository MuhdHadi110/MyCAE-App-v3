PROBLEMS?
☐ Backend won't start
  → Check Node.js App logs
  → Verify .env credentials
  → Click "Run NPM Install" again

☐ CORS errors
  → Node.js App → Restart
  → Check .env CORS_ORIGINS

☐ 404 errors
  → Check .htaccess in /public_html/
  → Check index.html is there

☐ Database errors
  → Double-check .env credentials
  → Verify user has ALL PRIVILEGES
