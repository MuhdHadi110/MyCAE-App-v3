========================================
TROUBLESHOOTING
========================================

Problem: Website shows "404 Not Found"
Solution:
- Check .htaccess file is in /public_html/
- Check index.html is in /public_html/

Problem: "Cannot connect to API" or CORS errors
Solution:
- Go to cPanel â†’ Setup Node.js App
- Click "Restart" on your app
- Wait 30 seconds and try again
- Check .env file has correct CORS_ORIGINS

Problem: Backend app won't start
Solution:
- Check Node.js App logs in cPanel
- Verify .env file has correct database credentials
- Click "Run NPM Install" again
- Click "Restart"

Problem: Database connection error
Solution:
- Double-check database credentials in .env file
- Make sure database user has ALL PRIVILEGES
- Database host should be "localhost"

Problem: White screen or blank page
Solution:
- Open browser console (F12)
- Look for errors
- Check if API URL is correct in requests
